# 排列组合

# 1. 计数原理

# 1.1 加法原理

定义 加法原理 又称为 分类计数原理。完成一个任务可以有 nnn 类办法，ai(1⩽i⩽n)a_i\left(1 \leqslant i \leqslant n\right)ai​(1⩽i⩽n) 代表第 iii 类方法的数目。那么完成这件事共有 S=a1+a2+⋯+an=∑i=1naiS = a_1 + a_2 + \cdots + a_n = \sum_{i=1}^n a_iS=a1​+a2​+⋯+an​=∑i=1n​ai​ 种不同的方法。

# 1.2 乘法原理

定义 乘法原理 又称为 分步计数原理。完成一个任务需要分 nnn 个步骤，ai(1⩽i⩽n)a_i\left(1 \leqslant i \leqslant n\right)ai​(1⩽i⩽n) 代表第 iii 个步骤的不同方法数目。那么完成这件事共有 S=a1a2⋯an=∏i=1naiS = a_1 a_2 \cdots a_n = \prod_{i=1}^n a_iS=a1​a2​⋯an​=∏i=1n​ai​ 种不同的方法。

# 2. 排列组合基础

本节 mmm 与 nnn 均为自然数，自然数被定义为大于等于 000 的整数。

# 2.1 排列数

定义 从 nnn 个不同元素中，任取 m(m⩽n)m\left(m \leqslant n\right)m(m⩽n) 个元素按照一定的顺序排成一列，叫做从 nnn 个不同元素中取出 mmm 个元素的一个 排列。

定义 从 nnn 个不同元素中取出 m(m⩽n)m\left(m \leqslant n\right)m(m⩽n) 个元素的所有排列的个数，叫做从 nnn 个不同元素中取出 mmm 个元素的 排列数，用符号 Anm\mathrm{A}^m_nAnm​（或者是 Pnm\mathrm{P}^m_nPnm​）表示。

根据乘法原理，对于 m⩽nm \leqslant nm⩽n 有排列数公式

Anm=n!(n−m)!=n(n−1)(n−2)⋯(n−m+1) \begin{aligned} \mathrm{A}_n^m &= \frac{n!}{\left(n-m\right)!} \\ &= n\left(n-1\right)\left(n-2\right) \cdots\left(n-m+1\right) \end{aligned} Anm​​=(n−m)!n!​=n(n−1)(n−2)⋯(n−m+1)​

如何推导

可以将这个过程看做需要 mmm 个步骤的任务，第一步可以选 nnn 个，第二步可以选 n−1n-1n−1 个，以此类推，第 mmm 步可以选 n−m+1n-m+1n−m+1 个，故得到公式 n(n−1)(n−2)⋯(n−m+1)n\left(n-1\right)\left(n-2\right)\cdots\left(n-m+1\right)n(n−1)(n−2)⋯(n−m+1)。

定义 全排列，当 m=nm = nm=n 时，所有元素都参与排列，其排列数为

Ann=n!1!=n! \mathrm{A}^n_n = \frac{n!}{1!} = n! Ann​=1!n!​=n!

规定：在不引起歧义的情况下，名词 排列、组合 与 排列数、组合数 的含义对应一致。

# 2.2. 组合

定义 从 nnn 个不同元素中，任取 m(m⩽n)m\left(m \leqslant n\right)m(m⩽n) 个元素组成一个集合，叫做从 nnn 个不同元素中取出 mmm 个元素的一个 组合。

定义 从 nnn 个不同元素中取出 m(m⩽n)m\left(m \leqslant n\right)m(m⩽n) 个元素的所有组合的个数，叫做从 nnn 个不同元素中取出 mmm 个元素的 组合数。用符号 Cnm\mathrm{C}^m_nCnm​ 来表示。

对于 m⩽nm \leqslant nm⩽n 有组合数公式

Cnm=n!m!(n−m)!=n(n−1)(n−2)⋯(n−m+1)m(m−1)⋯2⋅1 \begin{aligned} \mathrm{C}_n^m &= \frac{n!}{m!\left(n-m\right)!} \\ &= \frac{n(n-1)(n-2)\cdots(n-m+1)}{m(m-1)\cdots 2\cdot 1} \end{aligned} Cnm​​=m!(n−m)!n!​=m(m−1)⋯2⋅1n(n−1)(n−2)⋯(n−m+1)​​

如何推导

可以理解为排列的去重过程，排列是有序的而组合是无序的。

计算排列数可以得到 Anm=n!(n−m)!\mathrm{A}_n^m = \dfrac{n!}{\left(n-m\right)!}Anm​=(n−m)!n!​，进行去重，每次重复的数量即为 mmm 的全排列，除以 m!m!m! 即为结果。

组合数也被称为 二项式系数。组合数也通常记为 (nm)\dbinom{n}{m}(mn​)（LaTeX 符号 \binom{}{} 和 \dbinom{}{}），读作 nnn 选 mmm。

(nm)=Cnm \binom{n}{m} = \mathrm{C}_n^m (mn​)=Cnm​

特别地，规定当 m>nm > nm>n 时，Anm=Cnm=0\mathrm{A}_n^m = \mathrm{C}_n^m = 0Anm​=Cnm​=0。

组合数常见的性质：

  1. Cnm=Cnn−m\mathrm{C}_n^m = \mathrm{C}_n^{n-m}Cnm​=Cnn−m​
  2. Cn+1m=Cnm+Cnm−1\mathrm{C}_{n+1}^m = \mathrm{C}_n^m+\mathrm{C}_n^{m-1}Cn+1m​=Cnm​+Cnm−1​
  3. Cn0−Cn1+Cn2−Cn3+⋯+(−1)nCnn=0\mathrm{C}_n^0 - \mathrm{C}_n^1 + \mathrm{C}_n^2 - \mathrm{C}_n^3 + \cdots + (-1)^n\mathrm{C}_n^n = 0Cn0​−Cn1​+Cn2​−Cn3​+⋯+(−1)nCnn​=0
