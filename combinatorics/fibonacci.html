# 斐波那契数列

# 1. 定义

斐波那契数列（Fibonacci Sequence）的定义如下：

F0=0, F1=1, Fn=Fn−1+Fn−2, n⩾2 F_0 = 0,\, F_1 = 1,\, F_n = F_{n-1} + F_{n-2},\,n \geqslant 2 F0​=0,F1​=1,Fn​=Fn−1​+Fn−2​,n⩾2

该数列的值如下：

0, 1, 1, 2, 3, 5, 8, 13, 21, ⋯ 0,\,1,\,1,\,2,\,3,\,5,\,8,\,13,\,21,\,\cdots 0,1,1,2,3,5,8,13,21,⋯
