# 1. 公式和字体

数学公式排版将使用 LaTeX\LaTeXLATE​X，如果有其他可能的规范亦可能应用。

字体标准：

  1. 标准：部分变量或文本 abcd\text{abcd}abcd
  2. 斜体：标量、变量 a, b, ca,\,b,\,ca,b,c
  3. 粗体：常量或其他 C, R\mathbf{C},\,\mathbf{R}C,R
  4. 粗斜体：向量、矩阵 a, M\boldsymbol{a},\,\boldsymbol{M}a,M
  5. 空心体：部分数集 C, R\mathbb{C},\,\mathbb{R}C,R

# 2. 基本

  1. 空集：∅\emptyset∅
  2. 向下取整：⌊x⌋\lfloor x \rfloor⌊x⌋
  3. 向上取整：⌈x⌉\lceil x \rceil⌈x⌉
  4. 阶乘：n!n!n!
  5. 对数：log⁡m\log mlogm

如果对数不包含底数，这表明此处的底数不讨论。如果需要底数，若不说明，对数都是指 香农对数，即底数为 222 的对数。

求和符号有多种形式，

∑iai \sum_i a_i i∑​ai​

∑i=0nai \sum_{i=0}^n a_i i=0∑n​ai​

∑S⊆T∣T∣ \sum_{S \subseteq T} \left|T\right| S⊆T∑​∣T∣

# 5. 数论

LaTeX

在 LaTeX 数学公式中，竖线最好使用 \mid，因为 | 会产生不正确的间隔。

  1. 整除：x∣yx \mid yx∣y 表示 xxx 整除 yyy
  2. 取模：x  mod  yx \;\mathrm{mod}\; yxmody，也可能写为 x  %  yx \;\%\; yx%y
  3. 互质：x⊥yx \perp yx⊥y
  4. 最大公约数：gcd⁡(x, y)\gcd (x,\,y)gcd(x,y)，在无混淆意义的时侯可以写作 (x, y)(x,\,y)(x,y)
  5. 最小公倍数：lcm(x, y)\mathrm{lcm}(x,\,y)lcm(x,y)，在无混淆意义的时侯可以写作 [x, y][x,\,y][x,y]
  6. 组合数：Cnm\mathrm{C}_n^mCnm​，在无混淆意义的时侯可以写作下面的式子

Cnm=(nm) \mathrm{C}^m_n = \binom{n}{m} Cnm​=(mn​)
