# 1. 等差数列定义

定义 一般地，如果一个数列从第 222 项起，每一项与它的前一项的差等于同一个常数，这个数列就叫做 等差数列，这个常数叫做等差数列的 公差，公差通常用字母 ddd 表示，前 nnn 项和用 SnS_nSn​ 表示。

若 n⩾1, n⩾mn \geqslant 1,\, n \geqslant mn⩾1,n⩾m，则有等差数列的通项公式

an=a1+(n−1)d a_n = a_1 + \left(n-1\right)d an​=a1​+(n−1)d

或

an=am+(n−m)d a_n = a_m + \left(n-m\right)d an​=am​+(n−m)d

若 n⩾1n \geqslant 1n⩾1，则有等差数列前 nnn 项和求和公式

Sn=n(a1+an)2 S_n = \frac{n\left(a_1 + a_n\right)}{2} Sn​=2n(a1​+an​)​

或

Sn=na1+n(n−1)d2 S_n = na_1 + \frac{n\left(n-1\right)d}{2} Sn​=na1​+2n(n−1)d​
