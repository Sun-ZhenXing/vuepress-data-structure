# 均值不等式

下面讨论的内容均在实数范围内，不涉及复数或其他领域。

# 1. 基本不等式

基本不等式 是均值不等式的一个特例。

对于任意非负实数 a, ba,\, ba,b 有

(a−b)2⩾0 \left(\sqrt{a} - \sqrt{b}\right)^2 \geqslant 0 (a​−b​)2⩾0

那么有

a+b2⩾ab \frac{a+b}{2} \geqslant \sqrt{ab} 2a+b​⩾ab​

当且仅当 a=ba = ba=b 时取等号。

# 2. 平均数

在引出均值不等式之前，我们先引出几种常见的平均数。

定义 调和平均数 HnH_nHn​

Hn=n∑i=1n1/xi=n1/x1+1/x2+⋯+1/xn \begin{aligned} H_n &= \frac{n}{\sum_{i=1}^n 1/x_i } \\ &= \frac{n}{1/x_1 + 1 / x_2 + \cdots + {1}/{x_n} } \end{aligned} Hn​​=∑i=1n​1/xi​n​=1/x1​+1/x2​+⋯+1/xn​n​​

定义 几何平均数 GnG_nGn​

Gn=∏i=1nxin=x1x2⋯xnn \begin{aligned} G_n &= \sqrt[n]{\prod_{i=1}^n x_i} \\ &= \sqrt[n]{x_1 x_2 \cdots x_n} \end{aligned} Gn​​=ni=1∏n​xi​​=nx1​x2​⋯xn​​​

定义 算数平均数 AnA_nAn​

An=∑i=1nxin=x1+x2+⋯+xnn \begin{aligned} A_n &= \frac{\sum_{i=1}^n x_i}{n} \\ &= \frac{x_1 + x_2 + \cdots + x_n}{n} \end{aligned} An​​=n∑i=1n​xi​​=nx1​+x2​+⋯+xn​​​

定义 平方平均数 QnQ_nQn​

Qn=∑i=1nxi2n=x12+x22+⋯+xn2n \begin{aligned} Q_n &= \sqrt{\frac{\sum_{i=1}^n x_i^2}{n} } \\ &= \sqrt{\frac{x_1^2 + x_2^2 + \cdots + x_n^2}{n} } \end{aligned} Qn​​=n∑i=1n​xi2​​​=nx12​+x22​+⋯+xn2​​​​

# 3. 均值不等式

均值不等式定义为

Hn⩽Gn⩽An⩽Qn H_n \leqslant G_n \leqslant A_n \leqslant Q_n Hn​⩽Gn​⩽An​⩽Qn​

即

n1/x1+1/x2+⋯+1/xn⩽x1x2⋯xnn⩽x1+x2+⋯+xnn⩽x12+x22+⋯+xn2n \frac{n}{1 / x_1 + 1 / x_2 + \cdots + 1 / x_n } \leqslant \sqrt[n]{x_1 x_2 \cdots x_n} \leqslant \frac{x_1 + x_2 + \cdots + x_n}{n} \leqslant \sqrt{\frac{x_1^2 + x_2^2 + \cdots + x_n^2}{n} } 1/x1​+1/x2​+⋯+1/xn​n​⩽nx1​x2​⋯xn​​⩽nx1​+x2​+⋯+xn​​⩽nx12​+x22​+⋯+xn2​​​

证明均值不等式的方法有数学归纳法、琴生不等式、柯西不等式、构造几何法等方法，这里不再讨论，详细信息可以自行搜索。[1]

均值不等式最实用的推论是 n=2n = 2n=2 时的形式：

21/a+1/b⩽ab⩽a+b2⩽a2+b22 \frac{2}{1/a + 1/b} \leqslant \sqrt{ab} \leqslant \frac{a + b}{2} \leqslant \sqrt{\frac{a^2 + b^2}{2} } 1/a+1/b2​⩽ab​⩽2a+b​⩽2a2+b2​​

可以发现基本不等式是其中的一个不等式。
