# 柯西不等式

# 1. 柯西不等式表述

设 a∈R, i=1, 2, ⋯ , na \in \R,\, i = 1,\,2,\,\cdots,\,na∈R,i=1,2,⋯,n 则

(∑i=1naibi)2⩽∑i=1nai2∑i=1nbi2 \left(\sum_{i=1}^n a_i b_i\right)^2 \leqslant \sum_{i=1}^n a_i^2 \sum_{i=1}^n b_i^2 (i=1∑n​ai​bi​)2⩽i=1∑n​ai2​i=1∑n​bi2​

当且仅当 ai=kbia_i = kb_iai​=kbi​（kkk 为常数，i=1, 2, ⋯ , ni = 1,\,2,\,\cdots,\,ni=1,2,⋯,n）时取等号。

证明

要证柯西不等式，只要证

∑i=1nai2∑i=1nbi2−(∑i=1naibi)2⩾0 \sum_{i=1}^n a_i^2 \sum_{i=1}^n b_i^2 - \left(\sum_{i=1}^n a_i b_i\right)^2 \geqslant 0 i=1∑n​ai2​i=1∑n​bi2​−(i=1∑n​ai​bi​)2⩾0

只要证

(a12+a22+⋯+an2)(b12+b22+⋯+bn2)−(a1b1+a2b2+⋯+anbn)2⩾0 \left(a_1^2 + a_2^2 + \cdots + a_n^2\right) \left(b_1^2 + b_2^2 + \cdots + b_n^2\right) - \left(a_1b_1 + a_2b_2 + \cdots + a_nb_n\right)^2 \geqslant 0 (a12​+a22​+⋯+an2​)(b12​+b22​+⋯+bn2​)−(a1​b1​+a2​b2​+⋯+an​bn​)2⩾0

将上式拆项、整理得（此处省略拆项过程）

∑i=1n−1∑j=i+1n(ai2bj2+aj2bi2−2aiajbibj)⩾0 \sum_{i=1}^{n-1}\sum_{j=i+1}^n \left(a_i^2b_j^2 + a_j^2b_i^2 - 2a_ia_jb_ib_j\right) \geqslant 0 i=1∑n−1​j=i+1∑n​(ai2​bj2​+aj2​bi2​−2ai​aj​bi​bj​)⩾0

那么，只要证

∑i=1n−1∑j=i+1n(aibj−ajbi)2⩾0 \sum_{i=1}^{n-1}\sum_{j=i+1}^n \left(a_ib_j - a_jb_i\right)^2 \geqslant 0 i=1∑n−1​j=i+1∑n​(ai​bj​−aj​bi​)2⩾0

显然成立，当且仅当 aibj=ajbia_ib_j = a_jb_iai​bj​=aj​bi​ 成立，即 ai=kbia_i = kb_iai​=kbi​（kkk 为常数，i=1, 2, ⋯ , ni = 1,\,2,\,\cdots,\,ni=1,2,⋯,n）时取等号。

# 2. 重要结论

推论 1 设 ai∈Ra_i \in \Rai​∈R，则

∑i=1nain⩽∑i=1nai2n \frac{\sum_{i=1}^n a_i}{n} \leqslant \sqrt{\frac{\sum_{i=1}^n a_i^2}{n} } n∑i=1n​ai​​⩽n∑i=1n​ai2​​​

这个结论其实也是均值不等式的一个公式。

证明

令 bi=1/nb_i = 1/nbi​=1/n，那么柯西不等式变形为

(1n∑i=1nai)2⩽nn2∑i=1nai2 \left(\frac{1}{n}\sum_{i=1}^n a_i\right)^2 \leqslant \frac{n}{n^2} \sum_{i=1}^n a_i^2 (n1​i=1∑n​ai​)2⩽n2n​i=1∑n​ai2​

两边开根号得

∑i=1nain⩽∑i=1nai2n \frac{\sum_{i=1}^n a_i}{n} \leqslant \sqrt{\frac{\sum_{i=1}^n a_i^2}{n} } n∑i=1n​ai​​⩽n∑i=1n​ai2​​​

推论 2 设 xi∈R, yi>0x_i \in \R,\,y_i > 0xi​∈R,yi​>0，则

∑i=1nyi∑i=1nxi2yi⩾(∑i=1nxi)2 \sum_{i=1}^n y_i\sum_{i=1}^n \frac{x_i^2}{y_i} \geqslant \left(\sum_{i=1}^n x_i\right)^2 i=1∑n​yi​i=1∑n​yi​xi2​​⩾(i=1∑n​xi​)2

这是柯西不等式的分数形式，也是权方和不等式的一个特例。

证明

令 ai=xiyi, bi=yia_i = \dfrac{x_i}{\sqrt{y_i} },\,b_i = \sqrt{y_i}ai​=yi​​xi​​,bi​=yi​​，则柯西不等式变形为

结论得证。
